Call by value / Call by address

JS 中的數組是 Call by value，因此如果嘗試使用 = 進行複制時，只會複製到原始宣告的那組數組，而不是數組的值。要真正複製一個全新數組，需要在新的 value 下複製數組的值，這樣就不會引用內存中的舊 address。
